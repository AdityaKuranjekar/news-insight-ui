> **News-Insight AI** is a full-stack AI application that takes any news headline, classifies it into one of four categories using a **fine-tuned BERT model**, and then generates a **business intelligence insight** using a generative LLM (distilgpt2).

## ✨ Features

- 🔍 **Headline Classification** — Classifies any news headline into one of four categories: `World`, `Sports`, `Business`, or `Tech`
- 📊 **Confidence Score** — Shows how confident the model is in its prediction, with an animated progress bar
- 💡 **Business Insight Generation** — Generates a unique, LLM-powered insight for every headline
- ⚡ **Sample Headlines** — Six pre-built sample headlines to test the pipeline instantly
- ⌨️ **Keyboard Shortcut** — Press `Ctrl + Enter` (or `⌘ + Enter` on Mac) to analyze instantly
- 🌑 **Dark Glassmorphism UI** — Premium dark mode with animated neon orbs, glassmorphism cards, and micro-animations
- 📱 **Fully Responsive** — Works beautifully on both mobile and desktop

## 🤖 ML Pipeline

The AI core consists of two models working together in sequence:

### 1. Classifier — Fine-tuned BERT

| Property | Detail |
|---|---|
| **Base Model** | `bert-base-uncased` |
| **Task** | Text Classification (Sequence Classification) |
| **Dataset** | [AG News](https://huggingface.co/datasets/ag_news) |
| **Training Samples** | 2,000 (expandable) |
| **Eval Samples** | 400 |
| **Epochs** | 2 |
| **Precision** | FP16 (mixed precision) |
| **Optimizer** | Fused AdamW |
| **Labels** | `0=World`, `1=Sports`, `2=Business`, `3=Tech` |

### 2. Generator — distilgpt2

Takes the headline + its predicted category and generates a contextual business intelligence paragraph. No fine-tuning required — uses prompt engineering via a structured template.

**Prompt template used:**
```
News: {headline}
Category: {category}
Business Insight:
```

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Node.js 18+
- A GPU (recommended: Google Colab T4 for training)

---

### 1. Train the Model (on Google Colab)

```bash
# Install dependencies
pip install transformers datasets evaluate accelerate torch

# Run training (takes ~5-10 mins on T4 GPU)
python train.py
```

This will:
1. Load the `bert-base-uncased` tokenizer and model
2. Fine-tune it on the AG News dataset
3. Save the trained model to the `./model` directory

> 💡 **Colab tip:** Upload `train.py` to Colab, enable GPU runtime (`Runtime → Change runtime type → T4 GPU`), run the file, and then download the generated `model/` folder.

---

### 2. Run Predictions Locally

```bash
# Install dependencies
pip install transformers torch

# Run the predict script
python predict.py
```

**Example output:**
```
Headline  : Tesla launches new AI chip for self-driving cars
Category  : Tech (98.5% confidence)
Insight   : The acceleration of this technology stack signals a competitive moat...
```

---

### 3. Run the Frontend Locally

```bash
# Navigate to the frontend folder
cd frontend

# Install Node dependencies
npm install

# Start the dev server
npm run dev
```

Open your browser at **http://localhost:3000**

> ⚠️ The frontend currently uses a **mock API** that simulates the AI response. To connect it to the real `predict.py` backend, see the [Connecting the Backend](#-connecting-the-backend) section below.

## 🔌 Connecting the Backend

The frontend is ready to connect to a real backend. Inside `frontend/src/App.jsx`, find the `analyzeHeadline` function and replace the mock with a real API call:

```js
// BEFORE (mock):
async function analyzeHeadline(text) {
  return new Promise((resolve) => {
    setTimeout(() => resolve(pickMockResponse(text)), 2200)
  })
}

// AFTER (real API with FastAPI/Flask):
async function analyzeHeadline(text) {
  const res = await fetch('http://localhost:8000/predict', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ text }),
  })
  return res.json()
}
```

**Minimal FastAPI backend to wrap `predict.py`:**

```python
# api.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from pydantic import BaseModel
from predict import predict

app = FastAPI()

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_methods=["*"],
    allow_headers=["*"],
)

class HeadlineRequest(BaseModel):
    text: str

@app.post("/predict")
def analyze(req: HeadlineRequest):
    category, confidence, insight = predict(req.text)
    return {"category": category, "confidence": confidence, "insight": insight}
```

Run it with: `uvicorn api:app --reload --port 8000`
